## Easy Installation

1. Open the extensions sidebar on Visual Studio Code
2. Search for **Joker Smile**
3. Click **Install** to install it.
4. Click **Reload** to reload your editor.
5. Code/File Ôºû Preferences Ôºû Color Theme Ôºû **Joker Smile**.
6. üåü [Rate five-stars](https://marketplace.visualstudio.com/items?itemName=marcosvidolin.joker-smile#review-details).
